Completed · Review Action Multiplier

XP now scales because time passes.

The old XP-only multiplier has been replaced by an integer actions-per-review model. A successful card can now run 2, 3, or more normal game ticks, so XP, items, failures, side rolls, material consumption, and Utility batches all move together.

Runtime Flow

Answer cardEligible review enters `on_good_answer` once.
Read multiplier`ankiscape_review_action_multiplier`, falling back to the legacy key.
Run ticksGathering rolls N attempts; processing/Utility stop at depletion.
Aggregate feedbackFloating XP/activity popup appears once per review.
Undo snapshotOne saved rollback covers the whole multi-action review.
