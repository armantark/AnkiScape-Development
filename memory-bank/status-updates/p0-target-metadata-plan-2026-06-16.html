Guardrails

Behavior that must not move

No storage migration, no XP or item math changes, no action multiplier changes, no undo rollback changes, no Utility-as-skill registry move, no asset replacement, and no visual redesign. This is metadata extraction only unless tests expose a tiny compatibility fix.
