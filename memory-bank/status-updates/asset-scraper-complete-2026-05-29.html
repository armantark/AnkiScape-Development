Resolution

Maps display names and registry keys to wiki file titles and target paths.

2
Fetching

Queries OSRS first, falls back to RS3, sends a descriptive User-Agent, and retries with backoff.

3
Writing

Skips existing files unless forced, writes atomically, and can pad/resize to square transparent PNGs.

4
Provenance

Records source wiki, file page, image URL, sha1, timestamp, and license note per output path.
